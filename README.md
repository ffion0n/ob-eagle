# 支持批量本地转 eagle

支持一键将原有的本地附件 (图片/音频/视频) 批量导入 eagle 中，并替换成 eagle 链接，最后删除本地附件



# 增加音视频播放功能

Eagle 链接的音视频文件支持在 ob 中直接播放





# 支持手机查看附件

OB 手机端支持显示 eagle 链接的图片

在 termux 执行：
`python eagle附件脚本.py --root "/storage/emulated/0/eagle图库" --host 127.0.0.1 --port 6060`

下载 eagle-mobile 文件夹放到 `ob 仓库\. obsidian\plugins` 里
